"Inline-Block" without text-align is allow to create blocks horizontally but the contents inside of those blocks do not align along left, right side, or center of the blocks.

"Block with Centering" allows to create blocks vertically, each block will take space of full width of the page. Additional with margin element set to auto for left and right which makes the content stay in the middle of the page.

test

Block with text-align right is use to manipulate the text to align along the right side of the block.

Inline-block with TextAlign right is use to manipulate the text align with the right side of the blocks by using "text-align: right"

Slightly different from the inline-block with text-align:right. This tag uses text-align:center to make the contents align in the center of the blocks.

This particular section is using Float:right property instead of display property to make all the blocks float to the right side of the page.

This section is created by using Float:left property so the blocks are floating to the left side of the page.
